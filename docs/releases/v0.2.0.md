# Stenc v0.2.0

## Summary

Stenc v0.2.0 separates source JSON documents from generated static HTML/CSS
artifacts in target repositories.

## Added

- `stenc migrate`
- Generated artifact `.gitignore` policy
- Automatic static page regeneration in `open-docs.sh`

## Migration

From a target project root:

```bash
stenc migrate
./open-docs.sh
```

For a custom docs directory:

```bash
stenc migrate --docs-dir stenc
stenc migrate --docs-dir stenc --dry-run
```

`stenc migrate` removes generated pages from the Git index with
`git rm --cached`; local files remain on disk.
