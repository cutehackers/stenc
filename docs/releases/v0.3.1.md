# Stenc v0.3.1

## Summary

Release Stenc v0.3.1 with an installed-version check for the Stenc CLI.

## Changes

- Add `stenc --version`, `stenc -v`, and `stenc version`.
- Print the installed Stenc package version from the checked-out install
  source.
- Document version checking in the one-command install flow.

## Migration

No migration steps are required.
