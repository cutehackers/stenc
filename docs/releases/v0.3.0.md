# Stenc v0.3.0

## Summary

Release Stenc v0.3.0 with bounded rich supporting-section primitives for
Markdown-era expression while preserving JSON as the source of truth.

## Changes

- Add `paragraph`, `callout`, `quote`, `table`, `media`, `taskList`, and
  `diagram` blocks under `body.supportingSections[].blocks`.
- Add deterministic local media handling through `content/assets/` source files
  and generated `assets/` output.
- Render diagram blocks as escaped source panels only; no Mermaid runtime,
  scripts, CDN loading, or remote dependency is introduced.
- Update validator, renderer, rendered-page checks, templates, references,
  examples, and tests for the same source contract.

## Migration

No migration steps are required. Existing documents remain valid because rich
supporting blocks are optional.
